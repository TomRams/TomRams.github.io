Initial Design

As mentioned in the concept phase only the wings will be articulated and this phase focussed on deciding the overall method for moving the wings. Early on in this phase the goal to make the pterodactyl sit was removed, which means that the model will only be of a flying pterodactyl which can move its wings. Part of the reason this was removed was because of the design for the wing. While a real wing would have a joint in the middle of it where the hand is, this was removed because of its complexity for a prototype. This joint would have locked in flight so that wing worked properly, but the same effect can be achieved by removing it and using a fixed wing for the purposes of this project. The removal of this joint meant that it was no longer possible for the wing to adjust when sitting. The remaining wing joint is the joint closest to the body which can move the entire wing length.

The next stage was how the wing would move up and down using the remaining joint. During research, methods like hydraulics and pulley systems were considered to make the wings move. A hydraulic system was deemed too complicated and time consuming to make so it was not used due to the limited amount of time for this project. This meant a pulley system would be the system used and was the most suitable to move the wings. In a pulley system some form of rope slides over a low friction surface to redirect an object (EBSCO Information Services, Inc. | www.ebsco.com, 2019). An initial drawing was done to see how strings could be integrated into the arm and move it, like what is shown in Figure 4. In this drawing the red wire is being pulled towards the body to lift the wing up, and the blue wire is elasticated to pull the wing back down. The drawings included rollers for the string to move along as they would reduce the amount of resistance the string has when moving (shown in Figure 5), as well as eyes to keep the string in place (shown in Figure 6).

The arm would be fairly small, so the eye system would have to include small eyes so that they don’t extend too far away from the arm. This could result in the eyes breaking over time due friction or the arm and string bending in such as direction that the eyes break off. The roller system is more robust and has fewer things to go wrong. The roller system uses a number of rollers separated by a distance from the body and the string can go either on top or under the rollers, depending on how the rope has been attached.
