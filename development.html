Development

After the initial parts of the model were sketched out and the group was happy with the initial design, members of the group started to convert the design into a 3d model. For this 3d design software like Fusion360 or Blender could be used. Fusion is a computer aided design program (CAD) that is useful for mechanical part design as it can use precise measurements that the user inputs. These precise measurements are required in engineering as many models require calculations that are completely accurate so they can function correctly in the real world. It can also be used to simulate how a mechanical part would work in the real world including faults and stress points (3D-ace, 2019). As the focus of Fusion and other CAD programs is on mechanical parts, it is not as suited for creating models of organic objects like animals. This is where Blender is useful as it can be used for sculpting organic objects like a pterodactyl. Blender has many tools available for sculpting and designing organic shapes as well as some support for mechanical designs. While it is less precise, it can still be used for prototyping. Both pieces of software can simulate the movement of objects.

In this project only blender was used to design the model. This is because the main object is an organic shape, so is suited to software that can sculpt organic shapes. Also, as this is a prototype, exact measurements may not be required as much as a final version. Ideally, blender would be used to design the pterodactyl body and any other decorative parts and then Fusion is used to design the mechanical parts like connecting the wings together, hinges and rollers.

There are some other methods that could have been used to design the pterodactyl like a haptic pen. A haptic pen is a physical device where an object on a screen gives feedback to the pen. This allows for the user to feel the virtual model through the pen and can sculpt the model using it. This was not used during this project due to lack of experience with it in the group. It likely would have been a good tool to use as more detail could be applied to the model. The pen would only have been used for the sculpting part of the process and later the model would have been modified in a different program to create and add the wing mechanism.

Another method that could have been used to add more detail to the model would be 3D scanning. As pterodactyls would likely have scaly skin a piece of appropriate animal skin could have been acquired and this pattern applied to the model through scanning it. However, the model would have to be big enough to have this detail visible during the production stages. Additionally, this would likely use up a lot of the budget, and it was unknown whether there would be enough at this point.

During this stage members of the group started contacting technicians about getting prototypes 3D printed and about silicone to be applied to the outside. 3D printing is a good way of converting digital files to the real world. It can take an already existing 3D file in a variety of formats that have been designed in different programs (like Fusion and Blender) and convert it into a set of instructions for the 3D printer to use.

There are also different materials that can be used in this process. The most common materials are PLA, PETG and ABS and all three of these were considered for the project. PLA is commonly used in a variety of applications as it is a material that is easy to print. It is a rigid material so can be quite strong, however it also shatters relatively easily so would not be suitable for a primary school environment as sharp edges could lead to injuries or possibly more serious harm if it were ingested by a young child. It also has a low heat tolerance and becomes brittle and deforms with long exposure to sunlight. ABS is a strong and durable material that is frequently used for engineering products. It has a high heat tolerance but can be difficult to print with. ABS printing almost always requires an enclosure as it is prone to warping otherwise. It is also the most expensive of the three filaments on average. PETG is a good middle ground between PLA and ABS. It has a higher heat tolerance than PLA and is rigid and less prone to shattering with so is more ideal for schools (Wickstrom, 2025). The adhesion between layers is also good so it has harder to rip apart. It is the most resistant to sunlight so would be able to cope with long exposure to the sun. As a result, PETG was chosen for the final prototype and PLA was used for the test prototypes.

Silicone was decided to be used on the final prototype and final model as it would act as a skin. Silicone can look realistic and would be good for the scaly texture of the pterodactyl. It can also act as a barrier to stop fingers getting caught in the mechanism so that it is safer. A sheet of fabric would be used for the wings themselves as it is soft and flexible enough to not hinder the arms from moving.

While resin could be used for this project as it can be very detailed, it is likely too fragile to be near children. The resin would also have to be fully cured as it would be a health hazard otherwise, with children in particular likely to put it in their mouth. A hole would need to be added to drain any resin out from the process or else it would not cure properly.

While laser cutting wood would not be suitable for the project itself because of the materials needed, a wooden base to help keep the model steady could help. It would need to work with the 3d printed stand that has been designed so that the model has clearance. The main issue with laser cutting the stand would be that it needs to either be heavy or wide to ensure that the model doesn’t fall over. A heavy stand would take a number of passes for the laser cutter and take time to do, and the wide stand would be inconvenient to move and store.
